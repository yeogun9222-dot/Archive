대상자 imToken으로 $200 USDT 직접 송금 (눈 앞에서). 대상자가 직접 TUTU 입금 주소로 $200 전송 → Basic 등록 완료 → 소개 수당 권한 즉시 활성화 (White 달성).
3
증명 ①: 친구 1명 소개 → $20 ABA 즉시 입금 시연
에이전트가 씨앗의 추천으로 Flexible($100) 등록 → 씨앗 ABA에 $10 즉시 입금 (Basic 등록이므로 수당 권한 있음). 화면에서 실시간 확인. "Basic이어야 $10 들어오는 것" 명확히 전달.
4
증명 ②: ABA → ATM 현금 인출 — 실제 지폐를 손에
imToken USDT → Pursa 또는 Binance P2P → ABA 수령 → ATM 인출 전 과정 함께 진행. "디지털 숫자"가 "실제 지폐"가 되는 순간 신뢰 완성.
5
확산: 씨앗 → Telegram 그룹 → 자동 구축
성공 씨앗이 동료에게 자발 공유. 실제 ABA 입금 영수증·스크린샷 공유 → 가장 강력한 마케팅.
