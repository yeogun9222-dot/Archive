### **4️⃣ RewardsPage**

#### **4-1) The Sovereign Hall (상단 3개 카드)**

📍 **위치**: RewardsPage → The Sovereign Hall 섹션 → 상단 3개 카드

```
┌──────────────────────┐
│      [👑]            │  ← 왕관이 떠있는 상태 (Float 애니메이션)
│   [프로필 이미지]      │
│ Sovereign_Alpha      │
│ RED DRAGON SOVEREIGN │
└──────────────────────┘

위치: 프로필 이미지 위에 살짝 떠있는 상태 (겹치지 않음)
크기: 32px (모바일) ~ 48px (데스크톱)
색상: 금색 (Black Dragon용) / 빨강색 (Red Dragon용)
배경: 투명
애니메이션: Float 애니메이션 (위아래로 둥실둥실 떠다니는 효과, 2초 반복)
```

#### **4-2) RANKS탭 (랭킹 페이지)**

**위치 1: 회원 랭크 카드**

📍 **위치**: RewardsPage → RANKS 탭 → 회원 랭크 카드

```
[프로필 이미지]
[🔴 드래곤 아이콘 오버레이]
RED DRAGON
...

위치: 회원 프로필 이미지 위의 배지 형태 (오버레이)
크기: 14px ~ 20px
색상: 직급별 고유 색상 (진한 톤)
```

**위치 2: 섹션 헤더 (CURRENT IDENTITY STATUS)**

📍 **위치**: RewardsPage → RANKS 탭 → 섹션 헤더 (CURRENT IDENTITY STATUS)

```

색상: 직급별 고유 색상 (진한 톤)
```
