# Alpha Squad MCP 연동 — 팀원 간 협업 체계

## 핵심 원칙
- 모든 팀원은 [[../09_AI시스템_DevOps_Rex_Nova/jake-agent_인프라_개요|jake-agent]]를 통해 같은 DB(`chat_messages`)에 대화를 저장한다.
- 단체방(`/알파스쿼드`)에서 나온 발언은 각 팀원 개인 기록에도 동시 저장되어, 1:1 대화에서 맥락이 이어진다.
- 팀원 간 위임은 **반드시 `delegate_task` 도구를 실제로 호출**해야 한다. 말로만 "전달하겠다"고 하는 것은 금지 (허위 보고 방지 규칙, `personas.py`의 `_DELEGATION_RULE`).
- 위임 발생 시 텔레그램으로 대표님께 즉시 알림 (`notify_delegation`).

## 채널별 역할
| 채널 | 용도 |
|---|---|
| 텔레그램 개인 DM | 대표님 ↔ 제이크(또는 키워드로 다른 팀원) 1:1 |
| 텔레그램 그룹(플코아사랑방) | 그룹 멤버 ↔ 제이크, 호칭 자동 적용 |
| Claude Code 슬래시 명령어 | 대표님 ↔ 14인 개별 채팅방 (VSCode) |
| Claude Code `/알파스쿼드` | 전체 회의 |
| 이 채팅방(JAKE Brain) | 시스템/코드/인프라 총괄 — 일반 업무는 다루지 않음 |
