## 1. 왜 새 채팅방에서도 나를 기억하는가?

Claude Code는 두 가지 메커니즘으로 사용자를 기억합니다.

### A. Auto Memory 시스템 (핵심)
- 경로: `C:\Users\YG\.claude\projects\C--Users-YG\memory\`
- Claude가 대화 중 중요한 정보를 자동으로 `.md` 파일로 저장
- 매 대화 시작 시 `MEMORY.md` 인덱스를 자동으로 불러옴
- 새 채팅방을 열어도 이 폴더가 존재하는 한 기억이 유지됨

### B. settings.json (환경 설정)
- 경로: `C:\Users\YG\.claude\settings.json`
- 모델, 권한, 자동 업데이트 등 Claude Code 동작 방식 저장

### C. 대화 원본 (JSONL)
- 경로: `C:\Users\YG\.claude\projects\C--Users-YG\*.jsonl`
- 각 대화 세션이 UUID 형식으로 저장됨
- 백업본: `E:\Claude\ChatHistory\*.md` (매일 오전 6시 자동 백업)
