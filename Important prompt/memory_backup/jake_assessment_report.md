```

## 💡 제이크가 보고할 때 중점사항

### 성공 보고
```
"좋은 소식이 있어요.

1️⃣ 팀웍:
   - 에바, 사라, 미나가 완벽히 팀에 녹아들었어요.
   - 크로스펑셔널 협업도 3건 이상 성공했어요.

2️⃣ 프로덕션:
   - 이번 달 100명 배포를 완료했어요.
   - 성능도 예상치를 넘었어요.

3️⃣ 강점 보강:
   - 의사결정 속도가 3일 → 2일로 개선됐어요.

→ 결론: 팀이 중급에서 중상위급으로 성장하고 있어요."
```

### 문제 보고
```
"개선할 점도 있어요.

1️⃣ 프로덕션 경험:
   - 장애가 2건 발생했는데, 대응이 미숙했어요.
   - 모니터링 시스템을 보강해야 해요.

2️⃣ 도메인 지식:
   - 팀이 아직 LONGRISE를 깊이있게 이해 못했어요.

→ 조치: 다음 달에 __________할 예정입니다."
