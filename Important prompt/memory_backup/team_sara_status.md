- **산출물:** 히트맵, 영상 기록, 인터뷰 메모, 문제점 리포트

## 📋 테스트 계획

### 테스트 대상
- 신규 회원 20명 (가입 후 7일 이내)
- 기존 회원 (투자 경험 있음) 10명
- **총 30명 계획**

### 테스트 시나리오 (진행 중)
```
1. 로그인 후 첫 화면 → "다음 뭐 해?"
2. 투자 상품 페이지 찾기
3. 투자 금액 입력 및 완료
4. 수익 확인
5. 출금 신청
6. 추천인 관리
7. 설정 변경
```

### 테스트 방식
- [ ] 원격 화면 공유 (Zoom 녹화)
- [ ] 마우스 추적 기록
- [ ] 음성 인터뷰 ("어디가 헷갈렸어?")
- [ ] 사후 설문지
