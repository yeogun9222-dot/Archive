---

## 🎯 현재 미션

LONGRISE 투자 전환율 획기적 개선
- **문제:** 신규 회원이 로그인 후 투자까지 가는 비율이 낮음
- **목표:** 투자 전환율 **+30% 개선** (현재 기준점 설정 중)
- **수단:** A/B 테스트, 퍼널 분석, CRO 전문 지식

---

## 📊 현재 상황

### 기준점 설정 (진행 중)
```
필요 데이터:
□ 현재 투자 전환율은? (로그인 대비 투자 완료 비율)
□ 단계별 이탈률은? (어디서 가장 많이 떠나가나?)
□ 평균 투자 금액은?
□ 재투자율은?
□ 추천인 경로의 전환율은?
```

### 협력 팀원들과의 협의
- 바쿠(Data): 데이터 추출 및 분석
- 리리(Frontend): UI 변경 테스트 가능성
- 사라(Researcher): 사용자 행동 인사이트

---

## 🧪 계획된 A/B 테스트

### Test 1: CTA (Call To Action) 버튼 위치
```
A안: 현재 위치 (사이드바)
B안: 상단 고정 헤더 (눈에 띄기)

측정: 클릭률, 투자까지의 단계 수
목표: 버튼 가시성 +50% 향상
```

### Test 2: 투자 프로세스 단계 축소
```
A안: 현재 7단계 (상품선택 → 금액 → 확인 → 비밀번호 → 승인 → 완료 → 인증)
B안: 개선안 3단계 (상품 + 금액 일괄 선택 → 비밀번호 → 완료)

측정: 완료율, 이탈률, 평균 소요 시간
목표: 투자까지 2분 이내 단축
```

### Test 3: 페이지별 ROI 메시지
```
A안: 현재 (텍스트로만 수익률 표시)
B안: 시뮬레이터 추가 ($1000 투자 → 월 $X 수익)

측정: 투자 의사결정 시간, 투자 금액
목표: 의사결정 시간 -50%, 평균 투자액 +20%
```

### Test 4: 온보딩 플로우
```
A안: 현재 (투자 부터 시작)
B안: 개선안 (먼저 수익 구조 1분 영상 보기 → 그 후 투자)

측정: 첫 투자까지 소요시간, 투자 금액
목표: 신뢰도 증가로 투자액 +15%
```

## 📈 지금까지의 통찰

### 현재 데이터 (추정)
- 로그인 회원 중 투자하는 비율: **~5-10%** (추정, 바쿠와 확인 필요)
- 투자하려다 포기하는 비율: **~60%** (추정)
- 주요 이탈 지점: 단계 많음, 복잡한 설명, 불신

### 경쟁 분석 (유사 플랫폼)
- 일반적인 금융 앱 투자 전환율: 8-15%
- LONGRISE 목표: 12-20% (최소 2배 개선)
