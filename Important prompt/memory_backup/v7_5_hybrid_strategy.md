## 2️⃣ 중도해지 정산 (제인 담당)

### **V7.4 vs V7.5 비교**

| 항목 | V7.4 | V7.5 | 변화 |
|------|------|------|------|
| **수수료 구조** | 고정 (30→20→15%) | 동적 (배당차이 정액) | 유연성 ↑ |
| **최소환급금** | 없음 | 원금 × 15% 보장 | 안정성 ↑ |
| **회사 손실** | 불가피 | 통제 가능 | 재무 ↑ |
| **회원 신뢰** | 중간 | 높음 | 만족도 ↑ |

### **기간별 중도해지 수수료 (재정 기반)**
```
1~6개월: 원금 × 30%
  - 마케팅비: 17%
  - 카지노 계약 미달: 13%

7~9개월: 원금 × 20% (중기 운영비 감소)
10~11개월: 원금 × 15% (후기 운영비 최소화)
12개월 만기: 원금 × 10% (시스템 이용료)
```

### **15% 최소환급금 효과**

**예시 1: M6 해지 (Standard $500)**
```
기본계산: $500 - $280.80 - $150 = $69.20
최소보장: $500 × 0.15 = $75
최종환급: max($69.20, $75) = $75 ✓
```

**예시 2: M9 해지 (VIP $5,000)**
```
기본계산: $5,000 - $8,910 - $1,000 = -$4,910 (손실!)
최소보장: $5,000 × 0.15 = $750
최종환급: max(-$4,910, $750) = $750 ✓
회사 손실: 관리됨 (-$89 with floor)
```
