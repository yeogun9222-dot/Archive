## 🎰 2. 동적 수익률 시스템 (3요소 분해)

**일일 변동하는 수익률** = Pool + Betting + Futures

| 요소 | 평균 기여 | 설명 |
|------|---------|------|
| 🎲 Pool Size Factor | ~6% | 총 판돈 규모에 따른 하우스 우위 |
| 🎰 Betting Performance | ~8% | 4개 거점 베팅 성과 평균 |
| 📈 Futures Return | ~3% | AI 선물 거래 수익 |
| **합계 (VIP 기준)** | **~17%** | 월 평균 (변동폭 15~19%) |

**핵심**: 매일 다른 수익률이 계산되어 회원에게 표시됨

## 💰 3. CNYT 고정 비율 배분 (V7.4 핵심)

### A. 초기 CNYT 지급 (가입 시)
- **공식**: (원금 × 1%) ÷ CNYT 현재가
- **정책**: 즉시 락업 (M14 TGE까지 매각 불가, P2P 비용 $0)
- **공정성**: 가입 시점 무관 동일 USD 가치 보장

**예시 (VIP $5,000)**:
- M1 ($0.02): 2,500개 = $50 가치
- M5 ($0.04): 1,250개 = $50 가치
- M9 ($0.08): 625개 = $50 가치

### B. 월 CNYT 배당
- **공식**: (USDT 배당 × CNYT 비율) ÷ CNYT 현재가
- **정책**: 수령 후 7일 내 P2P 매각 가능, 7일 경과 시 자동 락업
- **P2P 비용**: 월 $978K 고정 (Reserve의 52.2%)
