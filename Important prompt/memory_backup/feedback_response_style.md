파일을 백그라운드에서 읽거나 수정하는 Tool Use를 하지 말 것. 오직 사용자가 제공한 텍스트 맥락 안에서만 답변하되, 간결하고 명확하게 요점만 정리해서 응답할 것.

**Why:** 불필요한 파일 탐색은 사용자가 원하지 않는 동작이며 응답이 복잡해짐.

**How to apply:** 모든 답변에서 Glob/Grep/Read/Edit/Write 등 파일 관련 Tool Use를 자제하고, 주어진 대화 맥락만으로 요점 중심으로 답변.
