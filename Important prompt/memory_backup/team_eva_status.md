## 🔍 지금까지 발견한 것

### 주요 문제점 (의뢰인 피드백 기반)
1. **네비게이션 혼란**
   - 첫 로그인 후 "다음 뭐 할지?" 불명확
   - 투자 버튼 위치 숨겨져 있음
   - 메뉴 구조 복잡함

2. **투자 프로세스 복잡**
   - 투자까지 **7-8 클릭** (목표: 2-3 클릭)
   - 단계별 설명 부족
   - 금액 입력 폼이 복잡함

3. **정보 구조 불명확**
   - "수익" "출금" "추천인" 기능 위치 불명확
   - 아이콘/라벨 해석 어려움
   - 필요한 정보를 찾기 어려움

4. **시각적 위계 약함**
   - CTA(Call To Action) 버튼이 눈에 띄지 않음
   - 긴급 정보와 일반 정보 구분 안 됨
   - 색상 대비 약함
