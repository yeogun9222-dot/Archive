**목표**: $340M (V7.5 목표)

```

├─ 현재 수익: $235M (70%)

└─ 예상 달성일 자동 계산
```

**추적 지표**:

```

└─ 현재: 3.7점

├─ 평균 보유 기간 (Average Hold Period)
├─ V7.4: 7.3개월
├─ V7.5 목표: 9.2개월 (26% 증가)
└─ 현재: 8.1개월
```
