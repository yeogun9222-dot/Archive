- **7일 강제 락업 시스템** 설계 및 구현
- **3단계 서킷브레이커 기능** 완벽 작동 보장

### **구체적 업무**

#### **A. CNYT 고정 비율 배분 시스템**

```
초기 CNYT 지급 (가입 시):
공식: (원금 × 1%) ÷ CNYT 현재가

구현 요구사항:
✅ VIP $5,000 가입 시:
   - M1 ($0.02): 2,500개 지급
   - M5 ($0.04): 1,250개 지급
   - M9 ($0.08): 625개 지급
   → 모두 동일 USD 가치 ($50) 보장

Admin 기능:
□ 회원별 초기 CNYT 자동 계산
□ 가입일자 × CNYT 가격 실시간 업데이트
□ 락업 상태 표시 (M14까지 매장 불가)
```

```
월 CNYT 배당:
공식: (USDT 배당 × CNYT 비율) ÷ CNYT 현재가

CNYT 비율:
- Basic: 2%
- Standard: 4%
- Premium: 6%
- VIP: 10%

Admin 기능:
□ 월 배당 시 자동 계산
□ 7일 카운트다운 타이머 표시
□ D-3, D-1 긴급 알림 발송
□ 7일 경과 시 자동 락업 처리
```

#### **B. 7일 강제 락업 시스템**

```
정책:
- 배당 수령 후 7일 내: P2P 매각 가능
- 7일 경과 시: 자동 락업 (M14까지 매장 불가)
- P2P 매각가: 배당 시점 가격 × 1.05

Admin 구현:
□ 배당 기록 자동 생성
□ 7일 카운트다운 Cron job (매일 자정 실행)
□ 일괄 락업 처리 스크립트
□ 로그 기록 (언제 누가 락업됐는지)

회사 효과:
□ 월 P2P 비용 고정 ($978K, 연 $11.74M)
□ Reserve의 52.2% 비용 극적 감소
```

#### **C. 3단계 서킷브레이커 시스템**

```
🟢 안전 (Reserve 30% 이상)
- 모든 수당: 100% USDT 지급
- Admin 표시: 초록색 배경, "안전 상태" 표시

🟡 주의 (Reserve 10~30%)
- 추천 수당: 70% 현금 + 30% 대기
- 기타 수당: 정상 지급
- Admin 표시: 노란색 배경, "주의 상태" 표시 + 경보 음

🔴 위험 (Reserve 10% 미만)
- 모든 수당: 50% CNYT 대체
- Admin 경고: 빨간색 배경, "위험 상태" + 긴급 경보
- 제이크에게 즉시 보고

Admin 구현:
□ Reserve 실시간 모니터링
□ 상태 변경 시 자동 알림
□ 지급 규칙 자동 전환
□ 변경 이력 기록 (감사 목적)
```

#### **D. 중도해지 정책**

```
환급 계산식:
환급액 = 원금 - USDT 배당 - CNYT 가치 - CNYT 매입비 - 기간별 수수료

기간별 수수료 (약정 상품):
- 1~6개월: 원금의 30%
- 7~9개월: 원금의 20%
- 10~11개월: 원금의 15%
- 12개월 만기: 0% (원금 90% 반환)

Flexible 상품:
- 수수료: 0% (손실 감안)
- 원금 반환: 100% 즉시 (24시간 내)

Admin 기능:
□ 중도해지 신청 자동 계산
□ 단계별 수수료 정확히 차감
□ 환급액 미리보기 제공
□ 수동 검증 옵션 (설리 QA용)
```
