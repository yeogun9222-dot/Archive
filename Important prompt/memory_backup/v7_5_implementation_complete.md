## 🎯 V7.5 3가지 전략 통합 상태

### A. Gemini 심리전 (마케팅) ✅
- **메시지**: "배당의 이득을 누리금이며, 원금은 언제든 인출 가능"
- **목적**: 회원 심리 안정화
- **효과**: 조기 환매 감소 → 자금 안정성 증가
- **구현 위치**: Dashboard.tsx 상단 (모든 관리자 즉시 확인)

### B. Claude 정산 (재무) ✅
- **실제 수익**: 환급액 = 원금 - 배당 - 운영비용
- **최소보장**: max(기본계산, 원금 × 15%)
- **목적**: 공정성 + 신뢰도 + 회사 손실 관리
- **구현 위치**: PayoutEngine.tsx (전체 정산 계산)

### C. Hybrid 효과 (통합) ✅
- **목표 수익**: $340M (V7.4 $300M → 13% 증가)
- **달성확률**: 75% (리스크-수익 균형)
- **실행 전략**: Gemini 심리 + Claude 정밀 정산
- **모니터링**: Dashboard 수익 캡 실시간 추적

```
✅ 수익 캡 Red Alert (80~90%+ 구간별 색상)

```
**파일 크기**: 약 +60줄 추가 (총 ~330줄)

### 2️⃣ PayoutEngine.tsx (핵심 로직)
```
✅ 15% 최소환급금 보장 계산
  - basicRefund = 원금 - totalExpense - earlyExitFee
  - minimumRefund = 원금 × 0.15
  - finalRefund = max(basicRefund, minimumRefund)

✅ 회사 손실 관리
  - companyLoss = 최소환급금 적용 시 손실액
  - 재무 추적 가능

✅ UI 표시
  - 기본 계산 vs 최소 보장 비교
  - 15% Floor 적용 전략 표시
```
**파일 크기**: 약 +40줄 추가 (총 ~180줄)

### 3️⃣ SystemSettings.tsx (가격 제어)
```
✅ CNYT 기본가 실시간 표시 ($0.01)
✅ 관리자 수동 가격 입력 ($0.01~$10.00)
✅ 범위 검증 + 경고 메시지
✅ 즉시 반영 버튼
```
**파일 크기**: 약 +65줄 추가

### 4️⃣ UserManagement.tsx (회원 관리)
```
✅ 4가지 정렬 옵션
  - 💰 잔액순 (기본)
  - 📊 활동순 (거래빈도)
  - 🔐 보안순 (OTP > 얼굴 > 휴대폰)
  - 🏆 직급순

✅ 보안 레벨 계산
  - OTP = 3점, 얼굴인증 = 2점, 휴대폰 = 1점

✅ 활동 점수 계산
  - 최근 거래 빈도 기반 (0~100점)
```
**파일 크기**: 약 +50줄 추가

### 5️⃣ WithdrawalDesk.tsx (해지 관리)
```
✅ 프로세스 가이드 (4 Step)
  - Step 1: 회원 중도해지 신청
  - Step 2: 시스템 자동정산
  - Step 3: 회원 출금 신청
  - Step 4: 관리자 승인/반려

✅ 관리자 권한 경계선
  - ✅ 할 수 있는 것: 출금 승인/반려
  - ❌ 할 수 없는 것: 중도해지 강제 실행

✅ 접기/펼치기 도움말
  - 모든 관리자가 프로세스 이해
```
**파일 크기**: 약 +80줄 추가
