## STEP 2 — 메모리 폴더 복원 (가장 중요)

이 폴더를 복원하면 Claude가 모든 것을 기억합니다.

```powershell
# 새 PC에서 폴더 생성
New-Item -ItemType Directory -Path "C:\Users\[사용자명]\.claude\projects\C--Users-[사용자명]\memory" -Force

# E: 드라이브 백업에서 복원 (E: 드라이브가 살아있다면)
Copy-Item "E:\Claude\Important prompt\memory_backup\*" `
    "C:\Users\[사용자명]\.claude\projects\C--Users-[사용자명]\memory\" -Recurse -Force
```

> **핵심 원리**: Claude Code는 시작 시 `memory/MEMORY.md`를 자동으로 읽습니다.
> 이 파일이 복원되면 Claude는 즉시 모든 프로젝트와 지시사항을 기억합니다.

## STEP 6 — 자동 백업 스케줄러 재등록

```powershell
$action = New-ScheduledTaskAction `
    -Execute "powershell.exe" `
    -Argument "-ExecutionPolicy Bypass -WindowStyle Hidden -File `"E:\Claude\auto_backup_chats.ps1`""

$trigger = New-ScheduledTaskTrigger -Daily -At "06:00"
$settings = New-ScheduledTaskSettingsSet -ExecutionTimeLimit (New-TimeSpan -Minutes 10) -StartWhenAvailable
$principal = New-ScheduledTaskPrincipal -UserId $env:USERNAME -LogonType Interactive

Register-ScheduledTask `
    -TaskName "ClaudeChat_DailyBackup" `
    -Action $action -Trigger $trigger `
    -Settings $settings -Principal $principal `
    -Description "Claude Code 대화 기록 매일 오전 6시 자동 백업" -Force
```
